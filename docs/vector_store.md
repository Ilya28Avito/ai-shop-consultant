# Vector Store — Qdrant

## Архитектура

Qdrant используется как векторная БД для семантического поиска по базе знаний
интернет-магазина ТехноМаркет. Коллекция `documents` содержит 54 чанка из 7
источников с payload-индексами для фильтрации.

## Конфигурация

| Параметр | Значение |
|----------|---------|
| Версия | qdrant/qdrant:v1.14.0 |
| Коллекция | documents |
| Размерность | 1536 (text-embedding-3-small) |
| Метрика | COSINE |
| HNSW m | 16 (default) |
| HNSW ef_construct | 100 (default) |

## Метрика: COSINE vs DOT

Проведено сравнение на 5 запросах из предметной области:

| Запрос | Cosine top-1 | Dot top-1 | Совпадение |
|--------|-------------|-----------|-----------|
| Сколько стоит iPhone 15? | iPhone 15 128GB — 89 990 руб. | iPhone 15 128GB — 89 990 руб. | ✅ |
| Как оформить рассрочку? | Для рассрочки нужен паспорт... | Для рассрочки нужен паспорт... | ✅ |
| Куда обратиться если сломалось? | Технически сложные товары... | Технически сложные товары... | ✅ |
| Бесплатная ли доставка в Москве? | Экспресс-доставка за 2 часа... | Экспресс-доставка за 2 часа... | ✅ |
| Кэшбэк на золотом уровне? | Уровень Золото: 7% кэшбэк | Уровень Золото: 7% кэшбэк | ✅ |

**Вывод:** OpenAI `text-embedding-3-small` возвращает нормализованные векторы
(L2-норма = 1), поэтому cosine similarity и dot product дают идентичное ранжирование.
Оставляем **COSINE** в production — это стандарт для семантического поиска,
понятный и воспроизводимый.

## Примеры фильтрации

### 1. Match по строке (category = "return")

```python
f = Filter(
    must=[FieldCondition(key="category", match=MatchValue(value="return"))]
)
```

Top-3 результата:
- [0.539] Вернуть товар можно в течение 14 дней...
- [0.535] Технически сложные товары можно вернуть только при дефектах...
- [0.525] Для возврата необходимо заполнить заявление...

### 2. Range по дате (последние 30 дней)

```python
f = Filter(
    must=[FieldCondition(
        key="created_at",
        range=DatetimeRange(gte=thirty_days_ago.isoformat())
    )]
)
```

Top-3 результата (все документы свежие — загружены сегодня):
- [0.539] Вернуть товар можно в течение 14 дней...
- [0.535] Технически сложные товары можно вернуть только при дефектах...
- [0.525] Для возврата необходимо заполнить заявление...

### 3. Композитный must + must_not

```python
f = Filter(
    must=[FieldCondition(key="category", match=MatchValue(value="catalog"))],
    must_not=[FieldCondition(key="source", match=MatchValue(value="promo_2025.md"))]
)
```

Каталог товаров без промо-документов. Top-3:
- [0.277] iPhone 15 Pro 256GB — 119 990 руб.
- [0.271] iPhone 15 256GB — 99 990 руб.
- [0.262] MacBook Air M2 256GB — 129 990 руб.

## Payload индексы

| Поле | Тип | Назначение |
|------|-----|-----------|
| source | KEYWORD | Фильтр по источнику документа |
| created_at | DATETIME | Фильтр по дате создания |
| category | KEYWORD | Фильтр по категории |
