# ИИ-консультант для интернет-магазина

## Тема
ИИ-консультант для интернет-магазина

## Описание
Проект представляет собой ИИ-ассистента, который помогает покупателям 
интернет-магазина получать ответы на вопросы о товарах, ценах и условиях 
доставки, а также сопровождает их на всех этапах выбора и оформления покупки.

## Почему эта тема
Работаю в сфере интернет-торговли и рекламы на Авито. Вижу, что большинство 
продавцов тратят много времени на однотипные вопросы покупателей. 
ИИ-консультант позволит автоматизировать этот процесс и повысить конверсию.
